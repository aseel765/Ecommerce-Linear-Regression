# Ecommerce Customers Analysis - Linear Regression

## 📌 Project Overview

This project analyzes an Ecommerce Customers dataset using Linear Regression to predict the yearly amount spent by customers based on their behavior.

## 📊 Dataset

The dataset contains customer information such as:

* Avg. Session Length
* Time on App
* Time on Website
* Length of Membership
* Yearly Amount Spent (Target Variable)

## ⚙️ Steps Performed

1. Data Loading and Exploration
2. Data Cleaning (handling unnecessary columns)
3. Feature Selection
4. Splitting Data into Training and Testing sets
5. Training a Linear Regression model
6. Making Predictions
7. Evaluating the model using:

   * MAE (Mean Absolute Error)
   * MSE (Mean Squared Error)
   * RMSE (Root Mean Squared Error)
8. Visualizing results (Actual vs Predicted)

## 📈 Results

The model achieved:

* MAE ≈ 8.5
* RMSE ≈ 10.4

This indicates a good predictive performance.
